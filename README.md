# Behavior Tree Mario

This is a program that creates a behavior tree agent that controls Mario using the MarioAI project.

## Usage

Run `ch.idsia.scenarios.Main.java`, which is configured to use the Behavior Tree agent.

## Dependencies

- `Java 11`

### Java Libraries

- `jdom`
- `asm-all`
- `junit4`
